A more customizable solution than the default wordpress tag cloud.

== Description ==

Options in widget: 

- Set number of tags to display
- Specify min and max posts to be displayed
- Specify font size ranges like a cloud
- Choose format for widget (list, flat, dropdown)
- Font size display unit (px, pt, em, percent)
- Enable/Disable Tags or Categories
- Show empty
- Display post count
- Sort by and Sort Order

== Installation ==

Estimated Time: 5 minutes

1. Upload the folder to the /wp-content/plugins/ folder on your site
2. In WP-Admin, you should see 'Tags' listed as an inactive plugin. Click the link to activate it. 
3. Once activated, go to Appearance > Widgets and Drop the Widget into your preferred sidebar
4. Once you've dropped in the widget, enter twitter handles (space separated) and click save.
